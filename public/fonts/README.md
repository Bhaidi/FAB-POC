# Graphik (self-hosted)

Licensed **Graphik** from Commercial Type / your enterprise font kit.

Bundled in this repo:

- `Graphik-Thin.ttf` → CSS weight **300** (light UI)
- `Graphik-Regular.ttf` → weights **400 / 500 / 600** (until Medium & Semibold files are added)

To improve hierarchy, add official web/TTF/WOFF2 cuts and extend `src` in `app/layout.tsx`, for example:

- `Graphik-Medium` → 500
- `Graphik-Semibold` → 600

Do not commit unlicensed font files.
